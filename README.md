# Shopware 5 dockerized

This is a production-ready dockerized setup for Shopware 5.

## Starting server

```bash
docker compose up -d
```

## Stopping server

```bash
docker compose down
```

## Installing Shopware

```bash
bash ./bin/install-shopware.sh sw5-shopware-1
```

## Updating Shopware

```bash
bash ./bin/update-shopware.sh sw5-shopware-1
```
